# A1Care Admin Panel

## Setup

1. Install dependencies:
   - `npm install`
2. Create env file:
   - copy `.env.example` to `.env`
3. Run:
   - `npm run dev`

## Roles

- `super_admin`: full access (admin management, audit logs, settings)
- `admin`: operational access (dashboard, services, doctors, bookings, users, notifications, reports)
